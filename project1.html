    How I made my website

    I had almost no previous experience on working with html5 so I started by reading up on the fundamentals on w3schools.com and started making a rough prototype. I wasn't particularly fond of the layout however and wanted it to look good on mobile devices so I decided to make use of a tutorial I found on Youtube. The underwhelming prototype can be seen below.

    I made an effort to divert from direct copying and used different tags and settings in my html script and .css file but the structure is at its core very similar. I omitted the use use of some inappropriate elements presented in the tutorial such as the newsletter subscription and made alterations to numerous visual elements.

    To begin with I installed brackets to write and test the html script and created a folder that included an html file, a folder for keeping images and a folder for the css file. In the file index.html I wrote the head first, beginning with declaring the type of script (html) and character encoding (utf-8). The meta name lines accomodate information about the viewport, description, keywords and the author of the webpage. Finally, the tab name is defined and the .css file is referenced as the stylesheet.

    With the head all sorted out I could begin with the body of the page where all of the text elements are contained in a coherent way (The div tag will be used frequently which will allow for styling by use of the .css file). The header was then defined which is the topmost part of the webpage and displays my name and three links: Home, Resume and VÉL403G (All list items). Note that the class is set as current in the list item Home as that is what the index.html provides information about.

    A poster section below the header was subsequently made solely for aesthetic purposes, where a container class includes a heading and a short paragraph

    To finish up with the front page I added some bulletpoints with pictures with similar methods described in the previous paragraphs.

    The majority of the "Resume" page is identical to the front page without the poster and icons. The short resume was written using simple paragraph elements and a cool sidebar was added to display some interests of mine.

    The "VÉL403G" page is very similar to the Resume page where links to individual projects are listed instead of the resume paragraphs.

    How I uploaded my website

    I followed the steps shown in this video which are the following:

    1. Install Gitbash.

    2. Right click inside the folder where all the website files are stored.

    3. In the command line, type git init and press enter to initialize an empty Git repository.

    4. In the command line, type git status and press enter to see the files contained in the folder.

    5. In the command line, type git add . and press enter to add the files to Git.

    6. In the command line, type git config --global user.email "your_email_address@example.com" and press enter.

    7. In the command line, type git config --global user.name "YOUR_USERNAME" and press enter.

    8. In the command line, type git commit -m "first commit" . and press enter to initialize an empty Git repository.

    9. In the command line, type git remote add origin GITHUB_REPOSITORY and press enter with your github repository link.

    10. In the command line, type git push -u origin master and press enter to push to Github. You will be prompted to login to your Github account.

    Finally click on your Github project link, go to settings and set the source to master branch. Your website URL will now be displayed and should be fully functional. In my situation however I had to add index.html to the URL for it to work.
